# Floral Engine

Projeto Vite + React + TypeScript com **@react-three/fiber**, **@react-three/drei** e **Framer Motion**. A cena traz uma flor 3D interativa em tela cheia.

## Como rodar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

## Deploy no Vercel (passo a passo)

1. Crie um repositório no GitHub e faça o push deste projeto.
2. Acesse [vercel.com](https://vercel.com) e faça login.
3. Clique em **Add New → Project**.
4. Selecione o repositório e confirme o framework como **Vite**.
5. Mantenha os padrões:
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
6. Clique em **Deploy** e aguarde a URL pública.
